# 日常異常事件流程圖素材

本資料夾保存日常異常事件流程圖文件與圖片素材。

| 檔案 | 說明 |
|---|---|
| `daily-incident-flow.html` | 日常異常事件可列印 HTML 原稿 |
| `inspection-flow.html` | 各類檢查可列印 HTML 原稿 |
| `assets/handler-line-chat.png` | 日常事件承辦 LINE 模擬對話圖 |
| `assets/supervisor-line-chat.png` | 日常事件主管 LINE 模擬對話圖 |
| `assets/handler-flow.png` | 日常事件承辦流程圖頁面 |
| `assets/supervisor-flow.png` | 日常事件主管流程圖頁面 |
| `assets/inspection-inspector-line-chat.png` | 各類檢查檢查人 LINE 模擬對話圖 |
| `assets/inspection-supervisor-line-chat.png` | 各類檢查主管 LINE 模擬對話圖 |
| `assets/inspection-inspector-flow.png` | 各類檢查檢查人流程圖頁面 |
| `assets/inspection-supervisor-flow.png` | 各類檢查主管流程圖頁面 |
| `screenshots/*.png` | 由目前公開網站擷取的實際畫面 |
| `../ISHA_日常異常事件流程圖_承辦主管.pdf` | 日常事件 PDF 輸出 |
| `../ISHA_各類檢查流程圖_檢查人主管.pdf` | 各類檢查 PDF 輸出 |

LINE 對話圖為教學模擬圖，不是 LINE 官方截圖。網站畫面截圖來自公開 GitHub Pages 或既有實際操作截圖。
