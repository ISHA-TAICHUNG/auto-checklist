# 日常異常事件流程圖素材

本資料夾保存日常異常事件流程圖文件與圖片素材。

| 檔案 | 說明 |
|---|---|
| `daily-incident-flow.html` | 可列印 HTML 原稿 |
| `assets/handler-line-chat.png` | 承辦 LINE 模擬對話圖 |
| `assets/supervisor-line-chat.png` | 主管 LINE 模擬對話圖 |
| `assets/handler-flow.png` | 承辦流程圖頁面 |
| `assets/supervisor-flow.png` | 主管流程圖頁面 |
| `screenshots/*.png` | 由目前公開網站擷取的實際畫面 |
| 本機 `docs/ISHA_日常異常事件流程圖_承辦主管.pdf` | 給使用者閱讀的 PDF 輸出；PDF 因 `.gitignore` 規則不發布到 GitHub Pages |

LINE 對話圖為教學模擬圖，不是 LINE 官方截圖。網站畫面截圖來自公開 GitHub Pages 或既有實際操作截圖。
